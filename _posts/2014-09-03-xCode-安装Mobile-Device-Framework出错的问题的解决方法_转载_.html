iTunes10.7更新，xCode 安装Mobile Device Framework出错的问题的解决方法

 

下午更新itunes10.7后，再次打开xCode时提示需要安装Mobile Device Framework，不过总安装失败，

提示" An unknown error occurred. See theinstall log for more details."

 

解决方法：

去finder中找到xCode文件，右键->"Show Package contents"->到"/Contents/Resources/Packages/"->安装“MobileDevice.pkg”->OK!
