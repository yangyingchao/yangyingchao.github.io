If you haven't noticed, quite a few online sharing and social networking sites are now supporting animated GIFimages, for example, on <a href="http://staff.tumblr.com/post/15623140287/1mb-gifs" target="_blank">Tumblr</a>, <a href="http://www.flickr.com/photos/markus-weldon-imagebank/4439159924/sizes/o/in/photostream" target="_blank">Flickr</a>, <a href="https://plus.google.com/communities/110524851358723545415" target="_blank">Google+</a>, and <a href="http://mashable.com/2013/08/29/gifs-return-to-facebook/" target="_blank">partly on Facebook</a>. Due to their ease of consumption and sharing, GIF-ed animations are now part of mainstream Internet culture.</p> <p>So some of you may wonder how you can create such animated GIFimages. There are various online or offline tools dedicated to create animated GIFimages. Another option is to create an animated GIF image off of an existing video clip. In this tutorial, I will describe <strong>how to convert a video file to an animated GIF image on <a href="http://xmodulo.com/recommend/linuxguide" style="" target="_blank" rel="nofollow" >Linux</a></strong>.</p> <p>As a more useful example, let me demonstrate how to <strong>convert a YouTube video to an animated GIF image</strong>.</p> <h3>Step 1: Download a YouTube Video</h3> <p>First, download a YouTube video that you would like to convert. You can use <a href="http://xmodulo.com/2013/03/how-to-save-youtube-videos-on-linux.html" target="_blank"><tt>youtube-dl</tt></a> tool to save a YouTube video as an MP4 file. Suppose you saved your favorite YouTube video as "funny.mp4".</p> <h3>Step 2: Extract Video Frames from a Video</h3> <p>Next, <a href="http://xmodulo.com/2013/06/how-to-install-ffmpeg-on-linux.html" target="_blank">install FFmpeg</a> on your Linux system, which I will use to extract video frames from the video.</p> <p>The following command will extract individual video frames, and save them as GIFimages. Make sure to use the output file format ("out%04d.gif") as is. That way, individual frames will be named and saved properly.</p> <code> ffmpeg -t &lt;duration&gt; -ss &lt;starting position in hh:mm:ss format&gt; -i &lt;input_video&gt; out%04d.gif </code> <p>For example, if you want to extract the video frames of input video, for 5 seconds, starting at 10 seconds from the beginning, run the following command.</p> <div class="console"> $ ffmpeg -t 5 -ss 00:00:10 -i funny.mp4 out%04d.gif </div> </p> <p>After FFmpeg is completed, you will see a list of GIF files created, which are named as "out[\d+].gif".</p> <h3>Step 3: Merge Video Frames into an Animated GIF</h3> <p>The next step is to merge individual GIF files into one animated GIF image. For that, you can use ImageMagick.</p> <p>First, <a href="http://ask.xmodulo.com/install-imagemagick-linux.html" target="_blank">install ImageMagick</a> on your Linux system if you haven't done so.</p> <p>Then, run the following command to merge created GIFimages into a single animated GIF file.</p> <code> convert -delay &lt;ticks&gt;x&lt;ticks-per-second&gt; -loop 0 out*gif &lt;output-gif-file&gt; </code> <p>In the command, "-delay" is an option that controls the animation speed. This option indicates that [ticks/ticks-per-second] seconds must elapse before the display of the next frame. The "-loop 0" option indicates infinite loops of animation. If you want, you can specify "-loop N", in which case the animation will repeat itself N times.</p> <p>For example, to create an animated GIF image with 20 frames-per-second and infinite loop, use the following command.</p> <div class="console"> $ convert -delay 1x20 -loop 0 out*.gif animation.gif </div> </p> <h3>Step 4 (Optional): Reduce the Size of an Animated GIF</h3> <p>The last (optional) step is to reduce the size of the created GIF file, by using ImageMagick's GIF optimizer.</p> <p>Use the following command to reduce the GIF size.</p> <div class="console"> $ convert -layers Optimize animation.gif animation_small.gif </div> </p> <p>Now you are ready to share the GIF image on your social networks.
