Some tips related to function , variable, symbol and symbol name.

1 How to check whether a function is defined? *fboundp*

1: (fboundp 'function-name)

2 How to check whether a variable is defined? *boundp

1: (boundp 'var-name)

3 How to Stringify a symbol, for example: 'aa –> "aa" ? *symbol-name*

1: (symbol-name 'aa)

4 How to make a named variable ? *intern*

1: (intern "aa")

5 How to check whether a feature is loaded? *featurep*

1: (featurep 'magit)
