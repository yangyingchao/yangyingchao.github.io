August 2014</span> </p> <p class="autor"><span>Tags:&nbsp;</span><a href="alexander.zeitler/tag/english/1" class="big-tag">english</a>,&nbsp;<a href="alexander.zeitler/tag/p4merge/1" class="big-tag">p4merge</a>,&nbsp;<a href="alexander.zeitler/tag/git/1" class="big-tag">git</a>,&nbsp;<a href="alexander.zeitler/tag/ubuntu/1" class="big-tag">ubuntu</a> </p> <article class="article"> <section class="content"><p>I’m using <a href="http://www.perforce.com/product/components/perforce-visual-merge-and-diff-tools">P4Merge</a> as Git merge and diff tool on all platforms.<br>This post describes how to install it on&nbsp;Ubuntu.</p> <p><span class="more"></span></p> <p>In order to set up P4Merge on Ubuntu, copy the extracted P4Merge download to <code>/opt/p4merge</code>:</p> <pre><code class="lang-bash">[~/Downloads/p4merge/p4v-<span class="number">2014.1</span>.<span class="number">888424</span>]$ <span class="built_in">sudo</span> mv * /opt/p4merge </code></pre> <p>Then symlink&nbsp;p4merge:</p> <pre><code class="lang-bash"><span class="built_in">sudo</span> ln <span class="operator">-s</span> /opt/p4merge/bin/p4merge /usr/local/bin/p4merge </code></pre> <p>After that, add this to your <code>~/.gitconfig</code>:</p> <pre><code class="lang-bash">[merge] keepBackup = <span class="literal">false</span>; tool = p4merge [mergetool] prompt = <span class="literal">false</span> [mergetool <span class="string">"p4merge"</span>] cmd = p4merge <span class="string">"<span class="variable">$<span class="caps">BASE</span></span>"</span> <span class="string">"<span class="variable">$<span class="caps">LOCAL</span></span>"</span> <span class="string">"<span class="variable">$<span class="caps">REMOTE</span></span>"</span> <span class="string">"<span class="variable">$<span class="caps">MERGED</span></span>"</span> keepTemporaries = <span class="literal">false</span> trustExitCode = <span class="literal">false</span> keepBackup = <span class="literal">false</span> [diff] tool = p4merge [difftool] prompt = <span class="literal">false</span> [difftool <span class="string">"p4merge"</span>] cmd = p4merge <span class="string">"<span class="variable">$<span class="caps">LOCAL</span></span>"</span> <span class="string">"<span class="variable">$<span class="caps">REMOTE</span></span>"</span> keepTemporaries = <span class="literal">false</span> trustExitCode = <span class="literal">false</span> keepBackup = <span class="literal">false</span> </code></pre> <p>Your next call of <code>git difftool</code> should look like this: <img src="../../images/p4merge.png" alt="P4Merge diff"></p> </section> </article> </div> </div> </div>
