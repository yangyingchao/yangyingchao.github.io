Unshared memory is reported as the USS (Unique Set Size). Shared memory is divided evenly among the processes sharing that memory. The unshared memory (USS) plus a process’s proportion of shared memory is reported as the PSS (Proportional Set Size). The USS and PSS only include physical memory usage. They do not include memory that has been swapped out to disk. Memory can be reported by process, by user, by mapping, or system-wide. Both text mode and graphical output are available.</p> <h3> Installation</h3> <p> To install smem<sup> [<a name="cite_ref2"> </a> <a href="#ref2" title="smem home page"> 2</a> ]</sup> type the following command under Debian / Ubuntu Linux:<br> <code> $ sudo apt-get install smem</code> <br> RHEL / CentOS Linux user type the following command:<br> <code> $ wget http://www.selenic.com/smem/download/smem-1.2.tar.gz<br> $ tar xvf smem-1.2.tar.gz<br>
