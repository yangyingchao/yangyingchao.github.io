# Emacs org mode hints – convert between csv & org table


本文为摘录(或转载)，侵删，原文为： https://gist.github.com/alesk/3826552

-   将 CSV 转换为 org 表格 <br />
    将文本导入文件，标记文本，然后按下 `C-c |` 。它会将该区域转换为表格。

-   要导出 <br />
    请使用 M-x org-table-export。
