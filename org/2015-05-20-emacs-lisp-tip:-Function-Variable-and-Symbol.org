Some tips related to =function= , =variable=, =symbol= and =symbol name=.

 - How to check whether a function is defined? =*fboundp*=
   #+BEGIN_SRC emacs-lisp -r -n
     (fboundp 'function-name)
   #+END_SRC

 - How to check whether a variable is defined? =*boundp=
   #+BEGIN_SRC emacs-lisp -r -n
     (boundp 'var-name)

   #+END_SRC

 - How to Stringify a symbol, for example: 'aa --> "aa" ? =*symbol-name*=

   #+BEGIN_SRC emacs-lisp -r -n
     (symbol-name 'aa)
   #+END_SRC

 - How to make a named variable ? =*intern*=
   #+BEGIN_SRC emacs-lisp -r -n
     (intern "aa")
   #+END_SRC
 - How to check whether a feature is loaded? =*featurep*=
   #+BEGIN_SRC emacs-lisp -r -n
     (featurep 'magit)
   #+END_SRC
