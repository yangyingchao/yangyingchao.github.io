Simply two tips:

1. List files in a  RPM package which is not installed:

   #+BEGIN_SRC sh -r -n
rpm -qlp xxxx.rpm
   #+END_SRC

2. List files in a RPM package which is installed:

      #+BEGIN_SRC sh -r -n
rpm -ql xxxx #with out extension name
      #+END_SRC
