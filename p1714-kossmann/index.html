3.1 Project Siberia

  • Siberia 是 Hekaton 项目用于管理冷数据的一个项目
    • Hekaton 则是 SQL Server 套件中的内存数据库
  • 查询处理

  • Hot Store

    • 用于查询热数据

  • Cold Store

    • 用于查询冷数据
    • 标准的 SQL Server 表

  • 冷热查询对客户端透明

    • 需要时候由 QueryProcessor 对两种数据的结果进行聚合操作

  • 冷数据的查询代价很高

  • filter 用于判定冷数据中是否有符合条件的数据

    • 返回 true: 可能有，需要访问冷数据
    • 返回 false: 肯定无，无需访问冷数据
