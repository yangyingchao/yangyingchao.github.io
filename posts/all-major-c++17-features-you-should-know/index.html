1.3 typename in a template template parameter 链接到标题

模板中以前只能用 class 来声明类型，现在可以用 typename 了

template <template <typename...> typename Container>
//            used to be invalid ^^^^^^^^
struct foo;

foo<std::vector> my_foo;

1.4 Nested namespace definition 链接到标题

Allows to write:

namespace A::B::C {
   // ...
}

Rather than:

namespace A {
    namespace B {
        namespace C {
            // ...
        }
    }
}
